Win10 始终以管理员打开命令行：
    Start -> Right Click -> More -> Open file location -> Right Click -> Properties -> Advanced -> Run as administrator 
    In this folder:
    C:\Users\Aaron Swartz\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\System Tools
   然后顺便可以创建快捷键，使用快捷键打开
   
   
conda config --show-sources 查看Anaconda下载源
conda config --remove-key channels 删除channels，就用默认源

conda 代理设置：修改 .condarc 文件  C:\Users\Aaron Swartz\.condarc
    proxy_servers:  
        http: http://localhost:1080
        
pip 代理设置：
    pip --proxy http://localhost:1080 install PACKAGE_NAME
    
pip install -r requirements.txt  pip通过文件安装包
pip install PACKAGE_NAME.tar.gz pip通过压缩文件安装包

pip设置下载源：
    临时：
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple some-package
    默认：
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

用youtube-dl 下载YouTube视频：
youtube-dl --proxy http://127.0.0.1:1080 --yes-palylist URL

cmd命令行关闭/开启路径：echo on/off
