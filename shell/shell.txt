第一个shell脚本：

	文件以 .sh 为扩展名

	第一行以 #！ 开头，声明所使用的shell的全路径

	注释以 # 开头

		#！/bin/bash
		# 打印输出hello,world!
		echo "hello,world!"

	脚本的执行需要一定权限：
		chmod +x SCRIPT_NAME #所有用户对该脚本有执行权限
		chmod 755 SCRIPT_NAME #用户所有者对该脚本具有完全的权限，其他人有执行权限


	运行脚本：
		进入所在目录
		./脚本名

变量的定义：
	myname='apple'  # 纯字符串
	yourname="banana" # 双引号内部会解析$和反斜杠
	time = `date` # 反引号执行系统命令

变量的引用：
	$VAR_NAME
	
	name="apple"
	echo $name
	echo ${name}
	变量名外加括号是为了帮助解释器识别变量的边界
	#!/bin/bash
	for skill in php java python c c++ linux
	do
		echo "I am good at ${skill}Script"
	done

	count=95
	echo 'The count value is $count.'
		- The count value is $count.
	echo "The count value is $count"
		- The count value is 95.

	time=`date`
	echo $time
		- xxxxx

查看所有环境变量： env
显示特定的环境变量的值：echo $PATH;echo $USERNAME


预定义变量：
	$# 命令行参数的数量
	$* 命令行参数的内容
	$? 命令执行后返回的状态, 0表示成功
	$$ 当前进程的进程号pid
	$0 当前执行的进程名
	$! 后台运行的最后一个进程号

echo:
	echo命令发送数据到标准的输出设备，数据采用的是字符串的方式，可以输出变量
	-e 参数可以识别转义字符
		echo -e "hello\tworld"
			- hello	world
	-n 忽略结尾的换行
		echo -e -n "hello\tworld"
			- hello	world【此行接着就是提示符】
